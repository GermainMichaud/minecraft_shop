# Minecraft Shop

> Application made with React (Typescript) and Redux for a technical test

## Clone repository

```sh
git clone git@github.com:GermainMichaud/minecraft_shop.git
cd minecraft_shop
```

## Install dependencies

```sh
npm i
```

## Run project in `DEV` mode

```sh
npm run dev
```

## Run tests

```sh
npm t
// or
npm test
```

## Build and run the project

```sh
npm run build
npm run preview
```
